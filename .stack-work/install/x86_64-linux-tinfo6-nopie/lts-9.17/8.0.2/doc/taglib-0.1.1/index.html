taglib-0.1.1: Binding to TagLib (ID3 tag library)

This is a Haskell binding to TagLib. TagLib is a popular library for reading and writing tags (ID3) of media files and getting audio properties from audio files. TagLib homepage: http://developer.kde.org/~wheeler/taglib.html
